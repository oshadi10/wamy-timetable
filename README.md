# WAMY Isiolo Timetable PWA

## 📱 How to Install on Your Devices

### Android Phone
1. Open Chrome and go to your GitHub Pages URL
2. Tap the **3-dot menu** (top right)
3. Tap **"Add to Home Screen"**
4. Tap **"Add"** — icon appears on your home screen!

### iPhone (iOS)
1. Open Safari and go to your GitHub Pages URL
2. Tap the **Share button** (box with arrow)
3. Scroll down and tap **"Add to Home Screen"**
4. Tap **"Add"**

### Windows PC
1. Open Chrome and go to your GitHub Pages URL
2. Click the **install icon** in the address bar (looks like a computer with +)
3. Click **"Install"**

---

## 🌐 How to Deploy to GitHub Pages (FREE)

1. Go to **github.com** and sign in (or create free account)
2. Click **"New Repository"**
3. Name it: `wamy-timetable`
4. Set to **Public**
5. Click **"Create repository"**
6. Upload these 3 files: `index.html`, `manifest.json`, `sw.js`
7. Go to **Settings → Pages**
8. Under "Source" select **"main"** branch
9. Click **Save**
10. Your app is live at: `https://YOUR-USERNAME.github.io/wamy-timetable`
